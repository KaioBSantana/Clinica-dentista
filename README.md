# 🦷 Clínica Dentista - Sistema de Gestão

Sistema web desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de gerenciar processos de uma clínica odontológica, facilitando o controle de pacientes e consultas de forma prática e organizada.

---

## 📌 Sobre o projeto

O sistema foi criado para simular um ambiente real de gestão clínica odontológica, permitindo o cadastro, edição, listagem e remoção de pacientes, além da organização de informações importantes para o atendimento.

Este projeto tem caráter acadêmico, com foco no aprendizado de desenvolvimento web e lógica de programação.

---

## 🚀 Funcionalidades

- Cadastro de pacientes  
- Listagem de pacientes cadastrados  
- Edição de dados de pacientes  
- Exclusão de registros  
- Organização de informações clínicas  
- Interface simples e funcional  

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  
- PHP (caso utilize no projeto)  
- MySQL (caso utilize banco de dados)
